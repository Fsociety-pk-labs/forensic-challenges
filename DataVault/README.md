# DataVault

## Challenge Name
**DataVault - Forensics (Medium)**

## Description
A sophisticated data breach has occurred at your organization. Multiple data sources have been compromised. You have recovered:
- **Database audit logs** - What data was accessed
- **Network traffic captures** - Data transfer patterns
- **Email communications** - Evidence of exfiltration
- **Access logs** - Who accessed what and when

Your task is to:
1. Reconstruct the complete attack chain
2. Identify the insider threat
3. Determine what data was stolen
4. Find how data was exfiltrated
5. Prove your case with timeline and evidence

## Difficulty
⭐⭐⭐ **Medium** (50-70 minutes)

## Learning Objectives
- Advanced multi-source forensic analysis
- Timeline correlation across systems
- Insider threat identification
- Evidence chain of custody
- Forensic report creation
- Complex incident investigation

## Files Provided
- `db_audit.log` - Database access records
- `network.log` - Network traffic summary
- `emails.log` - Email communications
- `access.log` - System access records

## Prerequisites
- Basic understanding of log formats
- Timeline analysis skills
- Ability to correlate events
- Critical thinking
- Report writing

## Real-World Scenario
This simulates actual insider threat investigations, data exfiltration cases, and multi-system breach analysis.
